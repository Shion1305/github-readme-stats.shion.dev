A stable, community-friendly public endpoint for `github-readme-stats` — operated by [@Shion1305](https://github.com/Shion1305).

## About

This is a public deployment of [anuraghazra/github-readme-stats](https://github.com/anuraghazra/github-readme-stats), offered as a stable alternative to the official endpoint (`github-readme-stats.vercel.app`) which can be flaky at times.

Just swap out the endpoint and you're good to go:

```text
https://github-readme-stats.shion.dev/api?username=Shion1305
```

## Reliability

This service runs on **Kubernetes** with **High Availability** in mind, so it stays up even when things get rough. The goal is to keep it running as a public resource for as long as possible.
